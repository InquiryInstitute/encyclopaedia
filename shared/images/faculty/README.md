# Faculty Portraits

This directory contains frontal bust portraits for faculty members.

## Image Requirements

- **Format**: PNG or JPG
- **Style**: Frontal bust (head and shoulders)
- **Size**: Recommended 400x500px or similar portrait aspect ratio
- **Background**: Neutral or transparent
- **Naming**: Use faculty ID format: `a.{surname}.png` or `a.{surname}.jpg`
  - Example: `a.james.png` for William James
  - Example: `a.peirce.png` for Charles Sanders Peirce

## Usage in Entries

Add to entry file:
```
:author-image: ../../../../shared/images/faculty/a.james.png
```

The portrait will appear next to the author signature at the end of the entry.

## Sources

Portraits should be:
- Public domain images
- Historical accuracy
- Dignified presentation
- Consistent style across all portraits
